# **App Name**: Dispatch Pro

## Core Features:

- Lead Capture: Collect potential client information via a web form and save it.
- Appointment Booking: Allow visitors to book consultation calls via a built-in calendar system.
- Service Showcase: Clearly outline dispatching services and benefits to potential clients.
- FAQ Generator: Use a tool to generate answers to common dispatching questions using AI. Input can be the details from the prompt, and answers should be geared toward prospective clients.
- Contact Options: Provide easy access to contact methods such as WhatsApp, phone, and email.
- Testimonial Section: Display quotes and success stories from happy clients.
- SEO Optimization: Optimize website structure with relevant keywords for better search engine ranking.

## Style Guidelines:

- Primary color: Dry Sage (#ccd5ae) for a gentle, earthy feel.
- Background color: Beige (#e9edc9) for understated sophistication and balance.
- Accent color: Light Bronze (#d4a373) for call-to-action buttons to draw attention with a touch of elegance.
- Utilize the default 'Inter' sans-serif font provided by Shadcn UI for a modern and objective feel. Consistent typography across the website.
- Leverage the iconography available within Shadcn UI, supplementing with logistics-related icons from a library like 'lucide-react' to visually represent services and benefits.
- Employ Shadcn UI's grid and layout components for a mobile-first, fully responsive design. Prioritize simple navigation and clear content hierarchy.
- Incorporate subtle transitions and animations using Shadcn UI's animation utilities or a library like 'framer-motion' for smoother user experience when navigating or loading content.
